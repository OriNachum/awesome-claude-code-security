# Security Policy

## Scope

This repository is a curated list of links and descriptions. It does not contain executable code.

## Reporting Issues

If you find that a linked resource has become malicious, compromised, or poses a security risk:

1. **Open a GitHub issue** describing the concern
2. **Include the URL** and the nature of the risk
3. We will review and remove or flag the resource promptly

For urgent concerns (e.g., a linked resource actively distributing malware), please email the maintainers directly.

## Link Verification

We periodically verify that linked resources:

- Are still accessible
- Have not been compromised or taken over
- Continue to meet our quality and safety standards

If you notice a suspicious link, please report it immediately.
